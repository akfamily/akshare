### 基金基本信息

接口: fund_name_em

目标地址: http://fund.eastmoney.com/fund.html

描述: 东方财富网-天天基金网-基金数据-所有基金的基本信息数据

限量: 单次返回当前时刻所有历史数据

输入参数

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

输出参数

| 名称   | 类型     | 描述  |
|------|--------|-----|
| 基金代码 | object | -   |
| 拼音缩写 | object | -   |
| 基金简称 | object | -   |
| 基金类型 | object | -   |
| 拼音全称 | object | -   |

接口示例

```python
import akshare as ak

fund_name_em_df = ak.fund_name_em()
print(fund_name_em_df)
```
