### 99 现货走势

接口: spot_price_qh

目标地址: https://www.99qh.com/data/spotTrend

描述: 99 期货-数据-期现-现货走势

输入参数

|-------|---------|----|
| 日期    | object  | -  |
| 期货收盘价 | float64 | -  |
